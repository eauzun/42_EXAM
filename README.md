# 42_exam

This repository contains the pool exams and all exam-rank exams used in 42 schools. Follow the steps below to run and evaluate the exams.

## Installation

To run this project, use the following command in your terminal to download and execute the necessary files:

```bash
bash -c "$(curl https://grademe.fr)"
Contents
This repository includes the following exams:

Pool Exams: Basic exams used during the initial phase at 42 schools.
Exam-Rank Exams: Advanced exams designed for students who have reached a certain level.
Usage
Open your terminal.
Run the command mentioned above to download and execute the exams.
The exams will be automatically evaluated.
